###Backend development

1.create git repo
    -git init

2. Add .gitignore file

3. Create .env file for environment variables & read data from .env with "dotenv" module
    -npm i dotenv
4. Generate the package.json
    -npm init -y

5. Create an express application
    -npm i express

6. Connect to database
    npm i mongoose

7. Add middlewares
    -body parsing middleware
    -error handling middleware

8. Design Schemas and create models

9. Design REST apis for all the resources
